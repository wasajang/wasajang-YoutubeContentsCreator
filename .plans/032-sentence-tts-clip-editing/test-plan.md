## 테스트 시나리오 (고객여정 기반)

### 시나리오 1: 나레이션 모드 음성 생성 (핵심)
```
홈 → 새 프로젝트(나레이션) → 대본 작성 → 음성 생성
```
- [ ] 대본 입력 후 TTS 생성 버튼 클릭
- [ ] 문장별로 개별 생성되는지 확인
- [ ] 각 문장에 audioUrl이 할당되는지 확인
- [ ] 문장별 타이밍이 정확한지 확인 (실측 기반)
- [ ] 총 재생 시간이 합리적인지 확인

### 시나리오 2: 문장별 재생 + 재생성
- [ ] 각 문장 옆 ▶ 재생 버튼 동작 확인
- [ ] 개별 문장 재생 시 해당 문장만 들리는지
- [ ] 🔄 재생성 버튼 → 해당 문장만 재호출
- [ ] 재생성 후 이후 문장 타이밍 자동 조정 확인
- [ ] 전체 미리듣기 동작 확인

### 시나리오 3: 편집 단계 나누기 (Enter 키)
```
음성 생성 후 → 씬 분할 → 이미지 생성 → 편집
```
- [ ] 편집 화면 진입
- [ ] 단어 칩 사이 gap 클릭 → "여기서 나누기" 표시
- [ ] Enter 키 → 나누기 실행
- [ ] Esc 키 → 나누기 취소

### 시나리오 4: 편집 용어 변경 확인
- [ ] 편집 화면에서 "씬" → "클립" 표기 확인
- [ ] 다른 단계(대본, 음성, 씬 분할)에서는 "씬" 유지 확인
