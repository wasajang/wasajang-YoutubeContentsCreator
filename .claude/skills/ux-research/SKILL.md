# UX 리서치 가이드

## 타겟 사용자 프로필

### 주요 페르소나
| | 초보 크리에이터 | 중급 크리에이터 | 파워 크리에이터 |
|---|---|---|---|
| **구독자** | 0~1K | 1K~100K | 100K+ |
| **영상 제작** | 처음 시작 | 정기 업로드 | 매일 업로드 |
| **기술 수준** | 편집 못함 | 기본 편집 가능 | 프리미어/에펙 사용 |
| **니즈** | "뭐라도 만들고 싶다" | "퀄리티 올리고 싶다" | "생산성 + 스타일 일관성" |
| **핵심 가치** | 쉬움 | 품질 | 속도 + 커스터마이징 |
| **가격 민감도** | 높음 (무료 선호) | 중간 | 낮음 (비용 < 시간) |
| **BYOK** | 사용 안함 | 관심 | 적극 사용 |

### 핵심 인사이트
- **초보자가 MVP의 핵심 타겟** — 진입 장벽을 극도로 낮춰야 함
- 중급자가 **유료 전환의 핵심** — Pro 플랜의 가치를 느끼게
- 파워 크리에이터는 **UGC 마켓의 공급자** — 프리셋 판매 동기 부여

## UX 분석 프레임워크

### 1. 사용자 여정 맵 (Journey Map)
각 단계에서 확인할 것:
- **행동 (Action):** 사용자가 뭘 하는가?
- **생각 (Thinking):** 뭘 기대하는가?
- **감정 (Feeling):** 만족? 혼란? 좌절?
- **기회 (Opportunity):** 어떻게 개선할 수 있는가?

### 2. 휴리스틱 평가 (Nielsen 10)
1. 시스템 상태의 가시성
2. 시스템과 현실 세계의 일치
3. 사용자 제어와 자유
4. 일관성과 표준
5. 오류 방지
6. 기억보다 인식
7. 유연성과 효율성
8. 미학적이고 미니멀한 디자인
9. 오류 인식, 진단, 복구
10. 도움말과 문서화

### 3. UX 문제 심각도
- 🔴 **Critical** — 사용자가 작업을 완료할 수 없음
- 🟠 **Major** — 작업은 가능하지만 상당한 혼란/불편
- 🟡 **Minor** — 사소한 불편, 개선하면 좋음
- 🟢 **Enhancement** — 현재 괜찮지만 더 좋아질 수 있음

## AntiGravity 핵심 플로우

### 행복 경로 (Happy Path)
```
HomePage → 시작점 선택 → IdeaPage (대본+스타일)
→ StoryboardPage (카드→컷→시드) → TimelinePage (편집→Export)
```

### 주의할 UX 포인트
1. **HomePage 3진입점** — 사용자가 차이를 이해하는가?
2. **IdeaPage 탭 전환** — SCRIPT/STYLE/CAST 탭이 직관적인가?
3. **StoryboardPage 4단계** — 단계 진행이 명확한가? 현재 위치를 아는가?
4. **TimelinePage Export** — 최종 결과물이 기대와 일치하는가?
5. **CastPage 카드 선택** — 5장 제한이 이해되는가?
